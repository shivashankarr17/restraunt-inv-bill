# restraunt-inv-bill
A program to create an invoice bill for a restaurant.
This program is designed to create invoice bills for a restaurant. It helps restaurant staff efficiently generate itemized bills for customers, making the checkout process smoother and more organized.
This respiratory consists of two folders namely; Code & Outputs. 
**Code Folder**: It consists of the code written in C programming for this project.
**Outputs Folder**: It consists of sample outputs of programming result.
